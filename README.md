# GooseBay-Generator

Weitere Informationen findet man im [Manual](doku/manual/main.pdf) und in der [Dokumentation](doku/main.pdf).

## Nutzung
Lokales JavaScript ausführen:
```bash
npm install
npm run start
```

Als Container ausführen:
```bash
docker-compose up
```
